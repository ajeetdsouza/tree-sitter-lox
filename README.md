# tree-sitter-lox

[Lox](https://craftinginterpreters.com/) grammar for
[tree-sitter](https://github.com/tree-sitter/tree-sitter).

## Local setup

Install `tree-sitter-cli`:

```bash
cargo install tree-sitter-cli
```

To generate bindings:

```bash
tree-sitter generate
```

To run tests:

```bash
tree-sitter test
```
